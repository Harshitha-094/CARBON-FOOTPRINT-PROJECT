# 🌱 Carbon Footprint Awareness Platform

## 📌 Project Description
This is a simple web-based Carbon Footprint Calculator that helps users estimate their monthly carbon emissions based on electricity usage and travel distance.

---

## 🚀 Features
- Calculates carbon emissions from electricity usage
- Includes travel distance impact
- Provides eco rating (Low / Medium / High impact)
- Simple and user-friendly interface

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 📥 How to Run the Project
1. Download or clone the repository
2. Open the folder in VS Code
3. Open `index.html`
4. Run using Live Server or open directly in browser

---

## 📊 Formula Used
- Electricity Carbon = kWh × 0.92
- Travel Carbon = km × 0.21
- Total = Electricity + Travel
